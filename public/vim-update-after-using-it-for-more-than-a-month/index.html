Vim update after using it for more than a month

09 May, 2014

#Tools #Vim

This post is a part of the Project Restore, with the primary aim of excavating my previous writings. The material contained herein may not hold practical relevance and is most likely completely useless.

It has now been close to a month since I started using Vim as my main editor. This post is about my Vim adventures.

Lately I have found watching videos to be an excellent source of learning new things. To increase my knowledge of Vim, I started by watching this excellent video titled Write code faster: expert-level vim by Ben Orenstein. Other than many cool tricks Ben shows in this video, the point which really hit home for me was to learn features in small proportions. For a complex editor such as Vim, I think this is an invaluable suggestion. I followed it and I can report positive results.

Next, I turned to the famous tutorials by Derek Wyatt. Following Ben Orenstien’s suggestion, I watched only two videos by Derek Wyatt:

  1. Basic Movement (Screencast 1)

  2. Basic Movement (Screencast 2)

The benefit I got out of watching these videos was that I found myself comfortable using Vim, enjoying it, and most importantly increasing my curiosity levels to learn more.

Many people have talked about the initial days or weeks of frustration. For me that period of frustration lasted few hours. Once I had a grip on basic movements, I was comfortable doing simple tasks.

I will keep posting my progress on Vim and anything interesting I find in Vim-land.
