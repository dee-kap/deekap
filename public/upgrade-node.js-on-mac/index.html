One way to upgrade node.js on your machine is to download the latest installer from node.js website. The other is to update node.js from CLI which is my preferred way. To upgrade node.js we should have n installed. npm install -g n Clean npm cache npm cache clean -f Install latest node.js stable version sudo n stable That’s all folks.
